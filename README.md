# 🎀 О проекте

**Snake Game** — это розовая версия классической игры «Змейка», созданная на игровом движке **Unity 6** с использованием языка **C#**.

Игроку необходимо:
- выживать как можно дольше
- собирать полезную еду
- избегать препятствий
- не сталкиваться с собой
- набирать максимальное количество очков
