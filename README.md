Snake Game
О проекте

Snake Game — это классическая игра «Змейка», реализованная на игровом движке Unity.
Главная цель игрока — продержаться как можно дольше, собирать полезную еду, избегать препятствий и набирать максимальное количество очков.

Проект сочетает классическую механику змейки с дополнительными игровыми элементами: несколькими типами еды, системой уменьшения змейки и статическими границами.

Суть игрового процесса
Продержаться максимально долго
Собрать как можно больше полезной еды
Избегать ядовитой еды
Не врезаться в препятствия и в саму себя
Сохранять жизнь змейке
Набрать максимальное количество очков
Получать удовольствие от игрового процесса
Технологический стек
Технология	Назначение
Unity 6	Игровой движок
C#	Язык программирования
Unity UI	Интерфейс и меню
Реализованный функционал
1. Базовая змейка
Классическая механика движения
Рост змейки после поедания еды
Проверка столкновений
Завершение игры при столкновении с собой
2. Разные виды еды
В игре реализовано несколько типов еды:
Обычная еда
Увеличивает счет
Увеличивает длину змейки
Бонусная еда
Дает дополнительное количество очков
Ядовитая еда
Уменьшает количество очков
Уменьшает длину змейки
Если остается только голова змейки — игра заканчивается
3. Границы игрового поля
Движение змейки ограничено игровыми границами
Генерация еды происходит только внутри игрового поля
При столкновении с препятствиями игра завершается
4. Дизайн
Реализована собственная цветовая гамма
Использован Unity UI для интерфейса и меню
Добавлено визуальное разделение типов еды

В проекте реализованы следующие алгоритмы:

Движение змейки

Уменьшение змейки
Спавн еды
Обнаружение столкновений
Управление
